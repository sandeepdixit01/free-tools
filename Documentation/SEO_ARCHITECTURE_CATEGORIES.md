## CROSS-CATEGORY STRATEGY

### Avoid Duplicate Content
Each category page has:
- **Unique H1** with category-specific keywords
- **Different benefits** tailored to user needs
- **Specific use cases** relevant to category
- **Unique guides** for that tool type

### Consistent Structure, Unique Content
- Same H2 framework across categories
- Different content under each H2
- Category-specific examples and use cases
- Tailored internal linking strategy

### Category Hierarchy
```
Homepage (free online tools)
```
